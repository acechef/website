MySQL Operator is a native Kubernetes MySQL management controller

You can run MySQL "statelessly" in cloud native environment, MySQL is a typical stateful software app, it's not easy to build a scalable and high available MySQL cluster in kubernetes environment. now with Grds MySQL Operator,you can run MySQL "statelessly" in cloud native environment just like your stateless apps.

 

Key Feature

  • Quickly build a containered MySQL Cluster or single instance,parameterizable cpu/memory/io configuration.
  • High-Availability automated inspect database failure,align and repair asynced binlog,elect new master instance.
  • Disaster Recovery backup task schedule，onekey media recover.
  • SlowSQL Analyst automated collect and analyze application SQL，supply optimization suggestions.
 

Features


Kubernetes native

MySQL Operator is a native Kubernetes MySQL management controller. This means you can easy to build a scalable and high available MySQL cluster in kubernetes environment.
